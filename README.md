# ModMenu.ExampleMod

This is an example mod for [ModMenu](https://github.com/MrJeranimo/ModMenu/tree/v0.1.0). It just serves to show how ModMenu works and to see the code behind it.

## Dependencies

1. ModMenu: [GitHub](https://github.com/MrJeranimo/ModMenu/tree/v0.1.0), [SpaceDock](https://spacedock.info/mod/4054/ModMenu)

## Building

You must copy over the `ModMenu.Attributes.dll` into the `ModMenu.ExampleMod` folder if you manually build this mod.

## Installation

Install the latest copy from releases and extract it to `Kitten Space Agency/Content/` then add the required info from the `mod.toml` into the `Document/My Games/Kitten Space Agency/manifest.toml`.
